# nodejs-web-server

### Run server

- `node src/app.js`

      	or

- `npx nodemon src/app.js`
- Enter url `localhost:3000` in your browser

### Stop server

- `ctrl + c`

### URL

- http://localhost:3000
- http://localhost:3000/weather
- http://localhost:3000/about.html
- http://localhost:3000/help.html
